# GestureImageCrop

使用QtQuick实现图片裁剪、手势缩放图片功能<br>


## Core
* `src/widget/photoview.h`
* `src/widget/photoview.cpp`
* `qml/widget/GestureCropView.qml`

## Platforms
- [x] Android
- [x] iOS

## Build
* Qt5.9.4
* Android
* Ndk r16b
* Gradle 3.4
